## mastra-opensearch-rag-userinsight-agent

This repository is a starter skeleton for a future project using OpenSearch and Retrieval-Augmented Generation (RAG).

### Structure

- `README.md`: Project overview and setup notes.
- `.gitignore`: Excludes build artifacts and sensitive files (such as environment variables and keys).

### Getting Started

1. Clone the repository:

```bash
git clone git@github.com:<your-username>/mastra-opensearch-rag-userinsight-agent.git
cd mastra-opensearch-rag-userinsight-agent
```

2. Create a `.env` file (not committed) for any local secrets or configuration.

3. Add your project code and update this README with more details as the project evolves.
